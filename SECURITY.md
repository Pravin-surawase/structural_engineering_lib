# Security Policy

## Supported Versions

This project is early-stage. Security fixes are provided for the latest `main` branch.

## Reporting a Vulnerability

If you believe you found a security issue:
- Do **not** open a public issue.
- Email the maintainer (use the email on the GitHub profile) with details.

Please include:
- Steps to reproduce
- Impact assessment
- Any suggested fix

We will acknowledge receipt and coordinate a fix/release.
