# Agent Roles

Defines 12 agent roles for the multi-agent collaboration framework.

## Roles

| Role | File | Scope |
|------|------|-------|
| Architect | `ARCHITECT.md` | Architecture decisions and design |
| Client | `CLIENT.md` | SDK and client interfaces |
| Dev | `DEV.md` | Core development |
| DevOps | `DEVOPS.md` | CI/CD and infrastructure |
| Docs | `DOCS.md` | Documentation |
| Governance | `GOVERNANCE.md` | Standards, quality, automation |
| Integration | `INTEGRATION.md` | System integration |
| PM | `PM.md` | Project management |
| Researcher | `RESEARCHER.md` | Research and analysis |
| Support | `SUPPORT.md` | User support |
| Tester | `TESTER.md` | Testing and quality assurance |
| UI | `UI.md` | Frontend and user interface |

Each role file defines responsibilities, capabilities, and workflow patterns.
