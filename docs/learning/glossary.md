# Glossary (Plain Language)

This is a simple glossary of terms used in the repo and outputs.

- Ast: Area of tension steel (mm^2)
- Asc: Area of compression steel (mm^2)
- Mu: Factored bending moment (kN-m)
- Mu_lim: Limiting moment capacity of section (kN-m)
- Vu: Factored shear force (kN)
- tau_v: Nominal shear stress (N/mm^2)
- tau_c: Concrete design shear strength (N/mm^2)
- tau_c_max: Maximum shear stress limit (N/mm^2)
- xu: Neutral axis depth (mm)
- xu_max: Maximum neutral axis depth (mm)
- pt: Percentage of steel (percent)
- Ld: Development length (mm)
- Lap length: Splice length for bars (mm)
- Serviceability Level A: Span/depth ratio check
- Serviceability Level B: Curvature-based deflection check
- BBS: Bar bending schedule (CSV list of bars and weights)
- DXF: Drawing exchange format (CAD output)
- Compliance report: Multi-check summary for a beam
- Governing case: The load case that controls design
- Schema version: Version of the output JSON structure
- Units: Explicit units expected at boundaries (IS456)
