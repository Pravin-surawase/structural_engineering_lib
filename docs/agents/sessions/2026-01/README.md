## Sessions Organization

Sessions are organized by:
1. **Month folder** (`2026-01/`, `2026-02/`, etc.)
2. **Agent prefix** (`agent-8-`, `agent-9-`, etc.)
3. **Type** (`weekN-completion-summary`, `weekN-plan`, etc.)

This allows:
- ✅ Easy chronological navigation
- ✅ Clear agent attribution
- ✅ Consistent naming across agents
- ✅ Scalable as more agents and months are added
